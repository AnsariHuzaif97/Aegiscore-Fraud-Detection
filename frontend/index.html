[commit: init] Data Ingestion & SMOTE

Raw transaction vectors are ingested. Severe class imbalances are algorithmically rectified using SMOTE to ensure the model correctly learns the minority fraud cases.

[branch: ml-core] XGBoost Inference Engine

Preprocessed data is routed into our highly optimized XGBoost pipeline. The model outputs a high-precision probability score, rapidly segmenting safe transactions from anomalies.

[merge: backend] FastAPI + SHAP Layer

A blazing fast asynchronous Python FastAPI backend orchestrates the logic. For anomalies, SHAP values are instantly computed to mathematically reverse-engineer the features.

[tag: production] Analytics Command Center

The frontend receives the payload and renders real-time data visualizations, dynamically rebuilding the Confidence Distribution histograms and logging the event.
